# Habit Tracker
A MERN stack habit tracking app designed to make building and managing habits simple.

# Built With
- React
- MongoDB
- Express.js
- Node.js
- Redux
  
# Features

### Authentication Features
- User authentication
- Hashed and salted passwords using bcrypt
- Protected Routes

### Habit Features
- Create habit
- Track habit completion
- Delete habit
- Streak tracking
- Heatmap visualization
